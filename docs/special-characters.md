# Special Unicode Characters

I can never find these characters to copy/paste so here they are.

- chevron right: 
- chevron left: 
- terminal icon: 
- circumscribed arrow: 
- vim icon: 
- git branch icon: 
- lock icon: 
- border characters: "─", "│", "─", "│", "╭", "╮", "╯", "╰"
